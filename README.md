# pythonprojects
A collection of small, beginner-friendly Python projects built for practice — covering fundamental programming concepts like control flow, functions, classes.

Projects Included
File	Description
calculator.py	A simple command-line calculator supporting addition, subtraction, multiplication, division, floor division, modulus, and exponentiation.
number_guess.py	A number guessing game where the player tries to guess a randomly generated number within a limited number of attempts.
cargame.py	A basic car simulation using state tracking (start, stop, quit) to mimic simple vehicle controls via user commands.
atm.py	An ATM simulator built with a Python class, demonstrating object-oriented concepts like encapsulation (private attributes) and user account handling.
temperature_converter.py	Converts temperatures between Fahrenheit and Celsius based on user input.
studentd_marks_analyzer.py	Takes marks for multiple students across 5 subjects and calculates total scores per student.

Concepts Practiced
Conditional logic (if/elif/else)
Loops (while, for)
Functions and nested functions
Classes and object-oriented basics
User input handling and type casting
Random number generation
